Overview

HTML stands for HyperText Markup Language and is used to structure web pages.

HTML Syntax

HTML uses tags such as <html>, <head>, and <body>.

Examples

Example of an HTML document:

<!DOCTYPE html>
<html>
    <head><title>My Page</title></head>
    <body>Hello World!</body>
</html>
